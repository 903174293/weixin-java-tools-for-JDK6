# Weixin Java Tools 
# 微信公众号/企业号开发Java SDK for JDK6

* 公众号（订阅号、服务号）：

maven：
```xml
<dependency>
  <groupId>com.github.binarywang</groupId>
  <artifactId>weixin-java-mp-jdk6</artifactId>
  <version>2.2.0</version>
</dependency>
```
gradle：
```groovy
compile 'com.github.binarywang:weixin-java-mp-jdk6:2.2.0'
```
